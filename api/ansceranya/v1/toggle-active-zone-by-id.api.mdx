<h1 className={"openapi__heading"}>Toggle Active status of a zone by ID</h1>

<MethodEndpoint method={"put"} path={"/zones/toggleActive/{zoneId}"}></MethodEndpoint>

<SecuritySchemes infoPath={"api/v1/anscer-anya-api"}></SecuritySchemes>



The Toggle Active Zone request allows you to toggle the _isActive_ property of a zone. This request takes in the zone's unique ID as a parameter, and will toggle the _isActive_ property of the zone.

**Setting the _isActive_ property to false will disable the zone in that map and Robot won't consider it for navigation purpose.**
