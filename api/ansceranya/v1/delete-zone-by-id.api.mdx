<MethodEndpoint method={"delete"} path={"/zones/{zoneId}"}></MethodEndpoint>

<SecuritySchemes infoPath={"api/v1/anscer-anya-api"}></SecuritySchemes>



The Delete Zone request allows you to remove a zone from the Map. This request takes in the zone's unique ID as a parameter, and will permanently delete the zone.

**In another case, you can have an toggleActive zone request that allows you to update the _isActive_ property of a zone. Setting the _isActive_ property to false will disable the zone in that map without deleting it completely.**
